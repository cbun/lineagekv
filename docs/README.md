# Research Notes

This directory contains public research notes for LineageKV. Local run memos
and one-off process notes are intentionally excluded from the public release.

## Entry Points

- `kv_cache_repair_technical_report.md` — consolidated method and result summary.
- `kv_memory_lineage_protocol.md` — memory-write, span, repair, and provenance contract.
- `kv_lineage_targeting_validity.md` — evidence that deployed targeters do not need hidden stale labels.
- `kv_address_content_causal_map.md` — value-only, key-only, and K+V causal matrices.
- `kv_repair_phase_diagram.md` — model/domain boundaries for choosing repair policies.
- `kv_memory_store_lineage_integration.md` — SQLite ledger prototype and runtime handoff.
